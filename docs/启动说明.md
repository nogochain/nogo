### 7. 停止节点

```powershell
Get-Process nogo -ErrorAction SilentlyContinue | Stop-Process -Force
```

### 8. 重新启动

```powershell
# 停止旧节点
Get-Process nogo -ErrorAction SilentlyContinue | Stop-Process -Force

# 等待几秒后重新启动
Start-Sleep -Seconds 2

# 启动新节点（使用上面的启动命令）
```

### 9. 数据目录

### 10. 注意事项

1. **端口占用**: 如果看到 `bind: Only one usage of each socket address` 错误，说明端口 8080 已被占用，需要先停止旧节点
2. **数据持久化**: 区块链数据会自动保存到 `./data/` 目录，重启后会继续之前的状态
3. **挖矿收益**: 挖矿奖励会自动发放到配置的矿工地址
4. **测试网**: 当前使用的是测试网配置（Chain ID=3），代币无实际价值
